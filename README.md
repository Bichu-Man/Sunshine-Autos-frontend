GTA-VICE-CITY aesthetics but in a car rental website(Sem 1 project)
Zero-dependency, vanilla architecture.

The Constraint: No JavaScript. No Frameworks. Pure HTML/CSS.

State Management: Implemented the Checkbox Hack to handle mobile navigation toggles. It uses the browser's native CSS engine rather than a scripting layer to manage UI states.

Fluidity: Utilized a CSS Grid auto-fit system for the fleet showcase, allowing for organic layout shifts without manual media-query stacking.

Depth: Applied background-attachment: fixed to create optical layering between the content and the horizon.
